# Info_spy
Esta ferramenta deve ser usada totalmente para fins educacionais ou profissionalmente como perícia ou investigação.

# Características de captura

- Data
- Hora
- Porcentagem de bateria
- IP Interno
- IP Público
- Imagem
- Localização

# Outras características 

- Redirecionamento
- Modelo de páginas prontas
- Salvamento de páginas

# Personalização de pré-visualização 

- Título
- Descrição
- Imagem

# 🛠 Instalação Kali/Debian

O comando termux-setup-storage precisa ser executado no termux antes de iniciar o kali

Execute os comandos abaixo,use o botão copiar à direita :

```bash
termux-setup-storage
```
```
sudo apt install python-is-python3
sudo apt install wget -y
sudo apt install git -y

git clone https://github.com/HiddenTechnology/Info_spy

cd Info_spy

wget https://github.com/HiddenTechnology/Info_spy/releases/download/Downloads/install.sh

chmod +x install.sh

./install.sh
```
```
python app.py
```
# 📱 Instalação Termux

```

termux-setup-storage

pkg install wget -y
```

# Instalação no Computador 💻

É preciso primeiro instalar o python no seu computador!

Depois baixe o repositório abaixo

https://github.com/HiddenTechnology/Info_spy/archive/refs/heads/main.zip

Extraia 

Abra a pasta, rode o terminal e execute o comando abaixo
```
py -m pip install flask requests pytz urllib3
```
